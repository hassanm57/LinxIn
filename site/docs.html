Your AI-run war room for the job hunt. Local-first, human-in-the-loop, built for Pakistan and remote-anywhere.

What is LinxIn

LinxIn turns your AI coding CLI (Claude Code and friends) into a private job-search system that runs entirely on your machine. It reads listings, scores them on Fit, Reality and Trust, flags scams and ghost jobs, and tailors your CV, cover letter and freelance proposals. It never applies for you: AI evaluates and drafts, you decide and submit.

Quick start

LinxIn is a set of skills plus a few small Node scripts. Clone it, install once, and drive it from your AI CLI.

# 1. get it
git clone https://github.com/hassanm57/LinxIn
cd LinxIn
npm install

# 2. onboard (in your AI CLI, e.g. Claude Code)
"set up my profile"

# 3. bring in jobs
"ingest this listing: <paste>"      # any board
npm run scan -- --keywords "backend" --remote-only   # ATS portals

# 4. evaluate → get the three scores + a report
"evaluate the new jobs"

# 5. for anything worth it
"tailor my CV and draft a cover letter for <job>"

# 6. browse it all
npm run dashboard        # http://localhost:4173

The three-score model

Every job gets three independent 0–100 scores. A role is worth pursuing only if it clears your thresholds on all three.

  • Fit — should you want it? Skills and stack match, seniority, mission, growth, company health, and comp vs. your target.
  • Reality — can a you-in-Pakistan land it and live on it? Geo-eligibility (the near-veto), timezone overlap, payment feasibility (Payoneer / Wise / Deel / EOR), and pay in PKR purchasing power.
  • Trust — is it legit? Verifiable company and recruiter, minus deductions for scam and ghost-job signals: upfront fees, WhatsApp-only recruiters, CNIC/bank asks before an offer, MLM framing, reposted-forever listings.

Defaults: Fit ≥ 65, Reality ≥ 50, Trust ≥ 70. A Trust score below 40 is quarantined as a likely scam.

The skills

profileOnboarding + your two CV bases (international, PII-stripped, and local).
ingestPaste any listing → a clean structured record in your pipeline.
scanPull fresh listings from Greenhouse / Ashby / Lever portals.
evaluateThe orchestrator: runs the three scores and writes a decision report.
geo-checkAre you actually eligible to be hired from Pakistan?
comp-realityWhat the pay is really worth in PKR, and how you'd get paid.
trust-checkScam and ghost-job detection, tuned for local fraud.
tailor-cvATS-clean CV tailored to one role → PDF.
cover-letterResearched letter that asks you for the human bits → PDF.
proposalFreelance track for Upwork / Fiverr / Toptal.
interview-prepLikely questions, company research, your best stories.
story-bankReusable STAR + Reflection stories the writing skills draw from.
trackKeeps the pipeline clean: dedup, status, integrity.

Privacy

Everything is local. Your profile, CVs and pipeline are plain-text files on your machine and are gitignored by default. The dashboard binds to 127.0.0.1 and never phones home. Nothing leaves your computer except the applications you choose to submit.

Contribute

LinxIn is open source (MIT) and built for people. The most valuable help right now is sharpening the scam-detection rules, geo-eligibility heuristics, and source recipes for local boards. Start with the repo and its CONTRIBUTING guide.
